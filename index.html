HTTP Request

GET https://levera/credits

Response Parameters

Parameter Type Description
id UUID Mã khoản vay
customer_name String Tên khách hàng
phone_number String Số điện thoại
email String Địa chỉ email
loan int Số tiền vay
loan_period String Thời hạn vay
shop_id int Mã cửa hàng của khách
id_card_images Array url ảnh CMT
household_book_images Array url ảnh sổ hộ khẩu
business_contract_images Array url ảnh ĐKKD
updated_at Time Thời điểm cập nhật cuối
inserted_at Time Thời điểm tạo
creator_id UUID Mã người tạo

date_start Time Thời điểm đơn hàng đầu tiên

HTTP Request

GET https://levera/analytic_detail/<id>

Query Parameters

Parameter Type Description
id UUID Mã khoản vay
split_by array Time.day: Thống kê theo ngày, Time.month: Thống kê theo tháng

Response Parameters

Parameter Type Description
Time.day string Thống kê theo ngày
Time.month string Thống kê theo tháng
returned string Đơn tính hoàn
success string Đơn tính thành công
discount int Chiết khấu
order_count int Số đơn hàng
price int Tổng giá
shipping_fee int Phí vẫn chuyển
